# FANCY HOME GOODS

A professional wed presence for **Fancy Home Goods**,a high class business in styling your home.


## Project Overview


This project is a website designed to showcase  the shops goods and how to purchase them as well.


## Project Structure
- `index.html` : this is the main page where you get to see our products as well as there prices.
- `style.css` : this is the page where our page is designed.

## Getting started
1. clone this repository on your machine.

```bash
git clone https://github.com/princemelz-creator/Fancy-home-goods.git

cd fancy-home-goods
```
2. Open `index.html` in any modern browser to view the web page.
3. No build tools or dependencies required for this version.

- **HTML5**: Semantic structure for accessibility and SEO.
